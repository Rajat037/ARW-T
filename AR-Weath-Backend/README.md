# A.R. Wealth & Tax Co. Backend

This is the standalone Express.js backend for the A.R. Wealth & Tax Co. platform.

## Features
- **AI Tax Assistant (Oreo)**: Direct integration with OpenAI GPT models for conversational Indian tax guidance.
- **Secure Authentication**: Express-session, password hashing, and CSRF protection.
- **Database Migrations & Setup**: Automatic tables initialization for PostgreSQL.
- **Secure API endpoints**: Handlers for contact forms, consultation scheduling, users profile, and Razorpay payment integration.

## Deployment
This backend is fully optimized for single-click deployment on platforms like **Railway** or **Render**, and links directly with hosted databases like **Neon**.

For full step-by-step deployment instructions, refer to the project's deployment guide.
